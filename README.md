# 🧠 Интелигентен Асистент (Intelligent Assistant)

Това е иновативно уеб приложение, базирано на архитектурата **Retrieval-Augmented Generation (RAG)**. Проектът позволява на потребителите да качват собствени документи и да водят интелигентен диалог с изкуствен интелект (Llama 3), който извлича точна информация директно от качените файлове и предоставя препратки към конкретни страници.

Този проект е разработен с фокус върху образователни цели и може да генерира автоматични тестове, флашкарти и резюмета от учебни материали.

## ✨ Ключови функционалности

*   **📄 Четене на множество формати:** Поддържа качване на PDF, DOCX, TXT и CSV файлове с прогрес бар в реално време (чрез WebSockets).
*   **⚡ Стрийминг в реално време:** Отговорите на асистента се генерират и изписват плавно на екрана (като ChatGPT), без да се блокира браузъра.
*   **🎯 Точни цитати и Native PDF Viewer:** AI асистентът посочва източника на своята информация. При клик върху източника, документът се отваря във вграден четец точно на съответната страница.
*   **🎓 Образователни инструменти:**
    *   💡 **AI Резюме:** Автоматично синтезиране на ключовите точки от избран документ.
    *   🧠 **Флашкарти:** Генериране на карти с въпроси и отговори за лесно запаметяване.
    *   📝 **Интерактивен тест:** Създаване на тест с избираеми отговори за проверка на знанията.
*   **🎙️ Гласови функции:** Вградено разпознаване на реч (Speech-to-Text) за задаване на въпроси и гласово четене (Text-to-Speech) на отговорите на български език.
*   **💾 История на разговорите:** Автоматично запазване на чат сесиите в локална база данни (SQLite).
*   **🎨 Модерен UI/UX:** Напълно адаптивен дизайн с поддръжка на **Тъмна тема (Dark Mode)**, Glassmorphism ефекти и плавни преходи.



## 🚀 Инсталация и конфигуриране

 Предварителни изисквания
1. Инсталиран **Python 3.10+**.
2. Инсталиран **Node.js 18+**.
3. Инсталиран **Ollama** на вашия компютър.
4. Изтеглен модел Llama 3. Отворете терминал и изпълнете:
   ```bash
   ollama run llama3


## Backend


# 1. Влизане в папката на бекенда
```bash
cd backend

# 2. Създаване на виртуалната среда (ще създаде папка venv)
python -m venv venv

# 3. Активиране на виртуалната среда (задължително за Windows!)
venv\Scripts\activate

# 4. Инсталиране на всички нужни зависимости
pip install -r requirements.txt


## Frontend

# 1. Влизане в папката на фронтенда
cd frontend

# 2. Инсталиране на зависимостите
npm install


## Стартиране на приложението

# Автоматично стартиране (Препоръчително)
1. Кликнете два пъти върху start.bat.

2. Скриптът автоматично ще отвори два терминала, ще активира виртуалната среда, ще пусне Python сървъра и ще стартира React приложението.

3. Отворете браузъра си на адрес: http://localhost:5173

# Ръчно стартиране
1. Терминал 1 (Бекенд):
cd backend

venv\Scripts\activate

uvicorn app.main:app --reload

3. Терминал 2 (Фронтенд):
cd frontend

npm run dev
